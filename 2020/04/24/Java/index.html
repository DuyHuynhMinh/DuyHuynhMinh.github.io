Primitive data types :

Type Description Set of Values
byte 8-bit signed two’s complement integer [-128 , 127]
short 16-bit signed two’s complement integer [-32768, 32767]
char 16-bit Unicode character
int 32-bit signed two’s complement integer [-2147483648, 2147483647]
long 64-bit signed two’s complement integer [-9223372036854775808,
9223372036854775807]
float 32-bit floating-point number (IEEE 754-1985)
double 64-bit floating-point number (IEEE 754-1985)
boolean a boolean value true or fasle

Note : We can use ‘object data type’.MIN_VALUE and ‘object data type’.MAX_VALUE to check range of primative data type. For example : Integer.MIN_VALUE and Integer.MAX_VALUE
